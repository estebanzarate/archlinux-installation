## Tools

- `git`
- `kitty`
- `neovim`
- [NvChad](https://nvchad.com/docs/quickstart/install/)
- `polybar`
- `type`
- `virtualbox-guest-utils`

```bash
sudo pacman -S git kitty neovim polybar virtualbox-guest-utils
```

```bash
sudo systemctl enable --now vboxservice
```
